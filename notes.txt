1. start backend:
    cd into backend
    1. Development:
        php artisan serve
    2. Production:
        execute "php artisan serve --host=0.0.0.0 --port=8000"
        open ngrok and run "ngrok http 8000"

2. start frontend
    cd into frontend
    1. development, no PWA
        npm start (development, No PWA)
    2. Production + PWA:
        npm run build
            npx serve -s build (local)
            upload "fyp/frontend/build" folder to netlify (cloud), then access to "https://ctj-fyp-webapp.netlify.app/"

3. create db tables
    cd into backend
    php artisan migrate

4. start facial recognition
    cd into backend/face/insightface
    .\venv\Scripts\Activate.ps1
    python test1.py

5. edit routes in "backend\routes\api.php"

    admin username: admin

7. if database goes boom:
    php artisan migrate:fresh --seed

InsightFace Benchmark Testing
    cd backend/face/insightface
        1. Activate the venv
            .\venv\Scripts\activate.ps1     (powershell terminal / visual studio code)
            .\venv\Scripts\activate.bat     (desktop access)

        2. Face recognition accuracy benchmark (requires LFW download above)
            python tests\benchmark_accuracy.py --limit 200    (quick, ~2-3 min)
            python tests\benchmark_accuracy.py                (full 6000 pairs, 15-30 min)

        3. Results saved to
            tests\results\accuracy_report.csv
            tests\results\score_distribution.png
            tests\results\roc_curve.png
            tests\results\stress_test_report.json
